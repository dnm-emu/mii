# МИИ — лабораторные работы

Автор: **Демченко Николай**

Данный репозиторий содержит набор лабораторных и практических заданий по курсу «Методы интеллектуального анализа».
Материалы структурированы по темам и предназначены для пошагового освоения основных подходов анализа данных и машинного обучения.

Каждая работа оформлена в виде Jupyter Notebook и может быть выполнена независимо при наличии необходимых библиотек.
